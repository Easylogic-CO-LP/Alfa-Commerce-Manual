# Plugin System Overview

Alfa Commerce is designed as a **marketplace platform** — third-party developers can extend it by writing plugins for payments, shipping, and form fields without modifying core code.

## Plugin Types

| Type | Group | Purpose | Example |
|------|-------|---------|---------|
| [**Payment**](./payment-plugins.md) | `alfa-payments` | Payment gateway integrations | Offline, card gateway, wallet |
| [**Shipment**](./shipment-plugins.md) | `alfa-shipments` | Shipping method integrations | Flat-rate, zone, courier, locker |
| [**Form Field**](./custom-field-plugins.md) | `alfa-form-fields` | Form field types | Color picker, file upload |
| [**Media**](./media-plugins.md) | `alfa-media` | Image processing (validate, resize, convert, thumbnail) | Resize, WebP, compress |
